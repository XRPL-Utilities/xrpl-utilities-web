Static summary (always available)

Pricing and finality

Each XR-Sentinel scan, each XR-Telemetry snapshot, and each XR-Pulse query is priced at $0.10 USD, payable in XRP. Drops are computed at request time from the live XRP/USD spot, so the advertised USD amount stays accurate as the spot moves. Successful payloads are paid in full; payment is final on settlement. If a request fails before settlement (verify-then-work-then-settle is enforced on all three services), no charge occurs and the presigned transaction is never broadcast. The 30-minute payment freshness window applies to web invoices; the 600-second window applies to x402.

What we provide

XR-Sentinel returns behavioral pattern classifications for XRPL accounts. XR-Telemetry returns an XRPL macro telemetry snapshot. XR-Pulse returns a public-source event feed with on-chain correlation against an institutional wallet watchlist. All three are best-effort, on-chain-derived data products. Sanity bounds and per-field provenance are declared in each service's /agents.json and /schema. Each field's value reads from a sanity-bounded cache, with the appropriate fallback applied when the cache is stale.

What we do not provide

None of these services produce risk scores, AML or sanctions determinations, compliance products, investment recommendations, trading signals, price predictions, or financial advice. The Burst Math output in XR-Telemetry is a sensitivity tool that shows what the math implies under explicit operator assumptions; it is not a forecast. Always do your own research before transacting.
